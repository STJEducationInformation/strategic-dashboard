Setup Guide

ขั้นตอนการติดตั้ง:
1. เปิด Google Sheet ใหม่ → Copy Sheet ID จาก URL
2. ไปที่ Extensions → Apps Script → วาง Code จากไฟล์ Code.gs
3. Deploy → New deployment → Web App → Execute as: Me → Who has access: Anyone
4. Copy Web App URL ใส่ในช่อง Script URL ด้านบน
5. กด บันทึกการตั้งค่า แล้วกด Initialize Sheet Structure
